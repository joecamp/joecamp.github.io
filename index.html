Projects

OCR (Java)
 This project on GitHub
Implemented an optical character recognition algorithm laid out in this paper.
Partners: Mark Barbera, Dennis Wessel, Nick Gerassimakis, Christopher Ware. Created for Professor Daniel Chester, using parts of his previous work as a framework.

MeggyJava Compiler (Java)
 Created a compiler for a simple object-oriented programming language, MeggyJava. Implemented scanning and parsing using JLex and JavaCup, implemented symbol tables, type checking, register allocation, and semantic checking in Java using the visitor design pattern. Presented compiler concepts at UD's CIS department showcase.
Partner: Ajith Vemuri.

Yard Wars (Java)
 This project on GitHub
An educational game created to promote sustainable landscaping habits. All art comes from collaboration with the University of Delaware's art department.
Partners: Shaun Repp, Kyle Leonard, Bryan Schmier.

CPU Scheduling Simulator (C)
 This project on GitHub
A program which reads a .txt file with lines of numbers mimicking processes running on a CPU. Several different scheduling algorithms can be chosen. Statistics are available after the simulation has run, such as average turn around time, max wait time, and average response time.

Vertiwords (C)
 This project on GitHub
A solution to a 2014 Mid-Atlantic Regional Programming Contest question. Takes in text in lines and rearranges the lines to spell out an input word vertically (within certain constraints).
