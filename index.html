CPU Scheduling Simulator (C)
 This project on GitHub

A program which reads a .txt file with lines of numbers mimicking processes running on a CPU. Several different scheduling algorithms can be chosen. Statistics are available after the simulation has run, such as average turn around time, max wait time, and average response time.

Vertiwords (C)
 This project on GitHub

A solution to a 2014 Mid-Atlantic Regional Programming Contest question. Takes in text in lines and rearranges the lines to spell out an input word vertically (within certain constraints).
